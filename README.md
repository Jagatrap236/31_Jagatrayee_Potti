## Agentic AI Travel Planner
An AI-powered travel planning web application that generates personalized, day-by-day travel itineraries based on a userâ€™s destination, travel dates, and interests. The goal of this project is to simplify trip planning by presenting all essential travel information in a clean and interactive interface.

## Project Overview
The application allows users to input their departure city, destination, travel dates, and interests. Based on this input, it generates a structured travel guide that includes:
- Best time to visit the destination  
- Estimated budget breakdown  
- Hotel recommendations with images, pricing, and ratings  
- Safety tips for travelers  
- Food suggestions  
- A detailed day-by-day itinerary  
The interface is designed to be visually appealing and easy to use, with real destination images and organized sections for better readability.

## Features
- Personalized Travel Itinerary: Generates customized travel plans tailored to user preferences.
- Destination Images: Displays real images of the destination to give users a visual understanding of the place.
- Hotel Recommendations: Shows recommended hotels with images, approximate pricing, ratings, and short descriptions.
- Structured Travel Guide: Information is clearly divided into sections such as budget, safety, food, and daily plans.
- PDF Download: Users can download the generated travel plan as a PDF for offline access.

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend:Python, FastAPI  
- Architecture: Frontend and backend are separated and communicate via API calls
